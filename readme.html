Implementation Guide

Setting Up Search Targets

  1. Navigate to the website containing the search tool you frequently use
  2. Click inside the search bar or input field
  3. Right-click and select "Add as Paste Target" from the context menu
  4. Assign a descriptive name that identifies the search destination (e.g., "Salesforce Accounts" or "Parts Catalogue")
  5. Configure auto-submit and tab reuse options based on your workflow requirements

Executing Searches

  1. Highlight relevant text on any webpage
  2. Right-click and navigate to "SearchSync"
  3. Select your configured search target from the menu
  4. The extension automatically opens the destination and executes the search

Troubleshooting

Context menu options are disabled or unavailable ▼
  • Right-click and select "Refresh" to update the extension's current state
  • Ensure you have clicked inside an input field before attempting to add it as a target
  • Verify that text is highlighted before accessing search targets
  • Reload the webpage if the extension appears unresponsive
  • Check that the extension is enabled in chrome://extensions/
Search target incompatible with specific platform ▼
  • Enterprise applications may implement security policies that restrict automated input
  • Disable auto-submit and manually execute searches as a workaround
  • Platform UI updates may require recreating the search target
  • Single sign-on (SSO) pages may require manual authentication before automated searches function
  • Contact your IT department if persistent issues occur with internal business applications
"Add as Paste Target" option not appearing ▼
  • Confirm you have clicked directly inside a text input field or search box
  • The element must accept text input (standard input fields, textareas, or contenteditable elements)
  • Use the "Refresh" option in the context menu to reset the extension state
  • Some shadow DOM implementations may prevent element detection
Configured search targets not displaying in menu ▼
  • Search targets only appear when text is selected on the current page
  • Verify targets are saved by clicking the extension icon to view all configurations
  • Reload the extension from chrome://extensions/ if synchronization issues persist
  • Check Chrome sync settings if configurations are missing across devices

Configuration Management

Viewing All Search Targets

Click the extension icon in your browser toolbar to access the management interface displaying all configured search targets.

Modifying Search Targets

  1. Click the extension icon to open the configuration panel
  2. Click on any search target to expand its settings
  3. Update the name, auto-submit preference, or tab reuse behavior
  4. Click Save to apply changes or Cancel to revert

Removing Search Targets

Expand the target in the management interface and click the Delete button. This action cannot be undone.

Security & Compliance

Enterprise-grade privacy and security

  • Zero data collection: No personal, business, or usage data is collected by this extension
  • No external communication: The extension does not transmit any information to external servers
  • Local storage only: All configurations are stored locally using Chrome's encrypted sync storage
  • No tracking or analytics: Your browsing activity and search patterns remain completely private
  • Open architecture: The extension operates entirely within your browser environment

All search target configurations are stored using Chrome's sync storage API and can synchronize across your authenticated Chrome browsers. Data remains encrypted and under your control at all times.
